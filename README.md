# 🚀 Mini LinkedIn Community Platform

A minimal yet powerful LinkedIn-like community platform where users can connect, share thoughts, and engage in professional conversations.

## ✨ Features

### 👤 User Authentication
- Register/Login with Email and Password
- Secure password handling
- User Profile with Name, Email, and Bio
- Profile Picture upload and edit

### 📝 Post Feed
- Create and share text-only posts
- View posts in a home feed with author’s name and timestamp
- Edit or delete own posts

### 🔍 Search & Discovery
- Search posts by keywords
- Search users by name or bio
- Filter posts using tags (optional)

### 💬 Engagement Tools
- Like/Unlike posts
- Comment on posts
- View all comments under each post
- Real-time interaction (optional with sockets)

### 👥 Following System
- Follow/Unfollow other users
- Personalized feed with posts from followed users
- View followers and following on profiles

### 🔔 Notifications
- In-app notifications for:
  - New followers
  - Post likes
  - Comments on your posts

### 👨‍💼 Profile Pages
- View any user's profile with their bio and posts
- See follower/following count
- Link to GitHub or portfolio (optional)

### 🎨 UI/UX Improvements
- Mobile-responsive layout
- Clean, modern design
- Dark mode (optional)
- Accessible components
